# Estudos com Pandas

## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar e consolidar meus conhecimentos em Pandas, uma das principais bibliotecas Python para manipulação e análise de dados.

Durante os estudos, explorei operações fundamentais para trabalhar com tabelas, organizar informações, realizar análises iniciais e visualizar dados de forma mais clara.

## Objetivo

O principal objetivo deste projeto foi aprender, na prática, como utilizar o Pandas para manipular dados de maneira mais eficiente, entendendo desde a criação de DataFrames até operações de análise e visualização.

## O que foi praticado

Ao longo do projeto, trabalhei com diversos recursos importantes da biblioteca, como:

- criação de DataFrames
- visualização das primeiras linhas
- visualização das últimas linhas
- aplicação de filtros
- identificação dos tipos de dados
- análise de quantidades
- cálculo de totais
- criação de novas colunas
- agrupamento de dados
- geração de gráficos

## Principais aprendizados

Com esse projeto, consegui praticar conceitos importantes para análise de dados, como:

- organização de dados em formato tabular
- leitura e interpretação de estruturas de DataFrame
- seleção e filtragem de informações
- transformação de dados
- sumarização de resultados
- agrupamento para análises comparativas
- visualização gráfica para apoiar entendimento dos dados

## Tecnologias utilizadas

- Python
- Pandas
- bibliotecas de visualização gráfica

## Observações

Este projeto foi importante para fortalecer minha base em análise de dados com Python, especialmente no uso do Pandas para exploração, transformação e visualização de informações.
